  1. User Key shows 'INSECURE'

    The length of the user id is the only thing that stops others from being able to impersonate you.

    If your user name is short then someone else - on purpose, or accidentally, could access your stats.

    To generate a new and secure user id please append `?pk=` to the url. After this `?pk=` part is not needed.

  2. Migrating user data

    To take your user id to another device (to get the stats):

    1. On the original device, go here.
    2. Copy the user data shown
    3. On the new device, add `?user_data=user_data_from_old_device` to url. This will save the data into a cookie and this needs to be done only once. After this `?user_data=` part is not needed.
